StyledString

class StyledString

A class that represents a string with visual styles.

Overview

This abstract base class provides methods with default implementations called styling methods, such as bold(), and background(). To use them, wrap your Python string with the s() function, followed by calls to one or more styling methods:

s('example').bold().background(Color.GREEN)

See Building Styled Text with NeonSign to learn more.

Printing a styled string

To print a styled string to the terminal, simply pass it to Python’s print() function:

print(
    s('example').bold().foreground(Color.GREEN)
)

Text styles

methodbold()

Increases the intensity of this string.

methoditalic()

Applies italic text style to this string.

methodunderlined()

Draws a horizontal line below this string.

methodcrossed_out()

Crosses out this string with a horizontal line through the center.

methodlight()

Decreases the intensity of this string.

methoddoubly_underlined()

Draws two horizontal lines at the bottom of this string.

methodoverlined()

Draws a horizontal line at the above this string.

methodframed()

Draws a frame around this string.

Layout

methodpadded([num_spaces])

Adds padding to both sides of the string.

methodpadded_left([num_spaces])

Adds padding to the left side of the string.

methodpadded_right([num_spaces])

Adds padding to the right side of the string.

Special effects

methodblinking()

Applies a blinking effect to this string.

methodhidden()

Rendering

propertyrendered

The rendered string containing the terminal commands that style this string according to the styling methods applied.

method__str__()

Renders this string as a Python string containing the terminal commands that style this string according to the styling methods applied.
