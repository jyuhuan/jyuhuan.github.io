function

styled(*contents)

styled(*contents: str | StyledString) → StyledString

Converts a string to a styled string object or creates a group of strings that can be styled together.

Parameters:

*contents – One or more Python strings, one or more styled strings, or a mix of both.

Returns:

A styled string object.

Raises:

TypeError – when contents contain an object that is neither a string nor a styled string.

Overview

To start adding styles to a Python string, wrap it with the styled() function. The function returns an instance of StyledString, which offers many styling methods such as bold() and color(). Use those methods to add styles to your string:

styled('example').bold().color(Colors.GREEN)

All styling methods are offered through the StyledString abstract base class. See its documentation or Building a styled string with NeonSign to learn more.

Printing a styled string

To print a styled string to the terminal, simply pass it to Python’s print() function:

print(
    styled('example')
    .bold()
    .color(Colors.GREEN)
)
