Color

class Color

A class that represents a color.

Overview

Anywhere NeonSign expects you to specify a color, you should provide an instance of Color.

Obtaining an instance

Use the class methods and constants on this class to obtain an instance. Do not call the initializer of any concrete color class directly.

Examples

  • Color.GREEN obtains the color green. The exact color value depends on the terminal and its configuration.

  • Color.color8(1) obtains the 8-bit color at index 1.

  • Color.rgb(255, 10, 20) obtains the 24-bit color, (255, 10, 20).

24-bit colors

methodrgb(red, green, blue)

Obtains the 24-bit color with the specified red, green and blue components.

methodhex(hex_string)

Obtains the 24-bit color with the specified hexadecimal value.

methodhsl(hue, saturation, lightness)

Obtains the 24-bit color with the specified hue, saturation and lightness using the HSL color model.

methodhsv(hue, saturation, value)

Obtains the 24-bit color with the specified hue, saturation and value using the HSV color model.
