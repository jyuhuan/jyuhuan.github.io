Colors

final class Colors

A class that provides access to color instances.

Important

Always use the constants and class methods in this class to obtain color instances. Do not call the initializer of SimpleColor, Color8 or Color24 directly.

Examples

  • Colors.GREEN obtains the color green. The exact color value is configured in the terminal.

  • Colors.color8(1) obtains the 8-bit color at index 1.

  • Colors.rgb(255, 10, 20) obtains the 24-bit color, (255, 10, 20).
